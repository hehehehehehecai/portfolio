**“打破传统玄学的单一文本，做现代年轻人的沉浸式心理暗示工具。”**

《赛博求签》是一款结合了传统东方玄学与赛博朋克视觉美学的跨平台应用。它打破了传统算命应用的单一文本生成模式，引入了物理引擎交互、本地数据持久化以及游戏化收集系统，致力于为现代年轻人提供沉浸式的心理暗示与趣味运势记录工具。

<br>

## ✨ 核心功能与产品亮点

* **沉浸式物理交互**：深度调用移动端硬件特性，利用 Accelerometer (加速度传感器) 结合物理算法，实现“真实摇晃”的求签反馈，拒绝单调的按钮点击。
* **赛博档案 (数据持久化)**：基于 AsyncStorage 构建本地运势数据库。应用能够自动记录、归档用户的历史求签数据，并提供多维度的数据分析与趋势展示，完成从“随机玩具”到“个人长线工具”的属性升级。
* **卡牌图鉴收集系统 (游戏化)**：创新性地引入 Gamification 机制，将签文按稀有度划分 (SSR/SR/R/N)，并配有动态点亮的赛博图鉴墙，大幅提升了应用的用户粘性与趣味性。
* **国际化双语支持**：内置完整的响应式中英双语切换逻辑，根据语言环境动态调整排版与日期格式。

<br>

## 🛠️ 硬核难点攻克

* **跨平台硬件 API 兼容性处理**：移动端代码（如加速度传感器监听器）在 Web 端运行时，会因浏览器缺少底层原生模块而导致致命的 TypeError 白屏崩溃。我引入了 `Platform.OS` 平台感知逻辑，在组件生命周期中针对 Web 环境进行硬件 API 的优雅降级 (Graceful Degradation)，彻底隔离原生模块，确保 Web 端的安全渲染与完美回退体验。

<br>

**技术栈**: `React Native (Expo)` | `AsyncStorage` | `Expo-Sensors` | `跨平台适配`
